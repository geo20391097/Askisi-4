## Περιγραφή εργασίας
Στην παρούσα εργασία πραγματοποιήθηκε προβολικός μετασχηματισμός (ομογραφία) στην ψηφιακή εικόνα "lavrio.jpg" με χρήση του πίνακα μετασχηματισμού H.

Η αρχική εικόνα μετασχηματίστηκε σε νέο γεωμετρικό σύστημα αναφοράς και έγινε αναγωγή (rectification) της εικόνας.

Για τον αντίστροφο μετασχηματισμό χρησιμοποιήθηκε ο αντίστροφος πίνακας της ομογραφίας και εφαρμόστηκε μέθοδος πλησιέστερου γείτονα (nearest neighbor interpolation) για την επανασύσταση της εικόνας.
